Linux per principianti

Passare a Linux sembra un salto nel vuoto solo la prima volta. In realtà, con la distribuzione giusta e pochi comandi base, ci si trova a proprio agio in pochi giorni.

Quale distribuzione scegliere

Una distribuzione ("distro") è una versione di Linux con un proprio sistema di installazione dei programmi, un'interfaccia grafica predefinita e un ritmo di aggiornamenti. Per chi arriva da Windows o macOS, tre opzioni si distinguono per facilità d'uso:

  • Linux Mint: interfaccia familiare, molto stabile, ottima per un primo approccio.
  • Ubuntu: la distro più documentata online, quindi facile trovare soluzioni ai problemi.
  • Fedora: più aggiornata, adatta a chi vuole software recente senza rinunciare alla stabilità.

Il terminale non è un ostacolo

Su Linux gran parte delle operazioni si può fare anche solo con il mouse, ma il terminale resta lo strumento più rapido per molte attività. Non serve impararlo tutto subito: bastano pochi comandi per orientarsi.

bash
pwd                 # mostra la cartella in cui ti trovi
ls -la              # elenca i file, inclusi quelli nascosti
cd Documenti        # entra nella cartella "Documenti"
mkdir progetti      # crea una nuova cartella
cp file.txt backup/ # copia un file in un'altra cartella

Installare programmi

A differenza di Windows, i programmi si installano quasi sempre tramite un "gestore di pacchetti" invece di scaricare file .exe da internet. Su Ubuntu e derivate, ad esempio:

bash
sudo apt update
sudo apt install nome-programma

sudo esegue il comando con permessi da amministratore; il sistema chiederà la tua password.

Il file system funziona diversamente

Non esistono lettere come C:. Tutto parte da una singola radice, indicata con /, e i dispositivi collegati vengono "montati" in sottocartelle di questa struttura.

Nota: la cartella personale dell'utente si trova di solito in /home/nome-utente, l'equivalente di "Documenti e Impostazioni" su Windows.

Primi passi consigliati

  1. Prova la distro da una chiavetta USB avviabile prima di installarla, senza toccare il sistema esistente.
  2. Fai un backup dei tuoi dati prima di qualsiasi installazione definitiva.
  3. Impara i comandi base elencati sopra: coprono la maggior parte delle esigenze quotidiane.
